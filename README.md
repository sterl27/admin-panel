# Admin Panel

A beautiful admin panel built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui components.

## Features

- **Dashboard**: Overview with key metrics
- **YouTube Module**: Manage and add YouTube videos
- **Blog Module**: Create and manage blog posts
- **Responsive Design**: Works on all devices
- **Modern UI**: Built with shadcn/ui components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/(admin)/` - Admin pages with sidebar layout
- `src/components/ui/` - shadcn/ui components
- `src/components/` - Custom components (sidebar, nav, etc.)

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons
